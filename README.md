# locfilm

### 🛠 Installation

```
npm install
```

### 🖥 Execution

**Development Environment**

```
npm run dev
```

> This project runs on **http://localhost:3000**

**Production Environment**

```
npm run build
npm run start
```

- React
- Sass
- ESLint
